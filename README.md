## Description

This is a Github application that creates a repository in authorized profile and adds a sample file to it. 
It uses Github OAuth and JWT for authentication. It needs 'repo' and 'workflow' permissions to execute the tasks.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
